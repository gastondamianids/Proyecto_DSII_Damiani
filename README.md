# Sistema Inteligente de Recomendación de Cultivos mediante Machine Learning

Proyecto desarrollado para la materia **Data Science 2**, enfocado en la aplicación de Machine Learning para recomendar cultivos según variables climáticas y del suelo.

---

# Objetivo

Desarrollar un modelo capaz de predecir el cultivo más adecuado utilizando variables como:

- Nitrógeno
- Fósforo
- Potasio
- Temperatura
- Humedad
- pH
- Precipitaciones

---

# Tecnologías Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

# Modelos Utilizados

- Regresión Logística
- Random Forest

---

# Resultados

| Modelo | Accuracy |
|---|---|
| Regresión Logística | 97.95% |
| Random Forest | 99.77% |

El modelo Random Forest obtuvo el mejor desempeño general.

---

# Principales Hallazgos

- Las precipitaciones y la humedad fueron las variables más importantes.
- Las variables climáticas y nutricionales mostraron alta capacidad predictiva.
- El proyecto demuestra el potencial del Machine Learning aplicado al agro.
